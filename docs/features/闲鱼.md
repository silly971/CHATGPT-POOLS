# 闲鱼功能启用指南

本功能包含：
- 管理后台「闲鱼订单」：配置 Cookie、同步订单、查看/清理订单
- 用户侧 `/redeem/xianyu`：输入邮箱 + 闲鱼订单号，自动匹配并完成兑换

## 1) 开启功能开关

- 进入管理后台：`/admin/settings`
- 在「功能开关」中启用「闲鱼（订单同步/兑换）」

禁用时相关页面与接口会返回 `403`（`FEATURE_DISABLED`）。

## 2) 配置闲鱼 Cookie

- 进入管理后台：`/admin/xianyu-orders`
- 粘贴有效 Cookie 并保存

系统可能会在同步过程中自动刷新 Cookie，并写回配置。

## 3) 配置兑换码库存（必须）

- 进入管理后台：`/admin/redemption-codes`
- 为渠道 `xianyu` 创建兑换码库存

## 4) 可选：开启 Cookie 自动续期调度

后端会按环境变量启动续期调度器（默认启用）：
- `XIANYU_LOGIN_REFRESH_ENABLED`：默认 `true`
- `XIANYU_LOGIN_REFRESH_INTERVAL_MINUTES`：默认 `30`

## 5) 验证

- 访问用户侧页面：`/redeem/xianyu`
- 输入邮箱 + 订单号
  - 若订单尚未入库：系统会使用已保存的 Cookie 拉取订单详情并写入数据库
  - 成功后会完成兑换并提示邀请状态
